## Usage
### Clone the repository
    git clone https://github.com/gw-zhong/MulDeF.git
### Download the datasets
- [BaiduYun Disk](https://pan.baidu.com/s/1ZKVXD1lVr5iYXY2yqJYwMw) ```code: mp9o```

### Run the code
 ```python
python train.py
 ```
